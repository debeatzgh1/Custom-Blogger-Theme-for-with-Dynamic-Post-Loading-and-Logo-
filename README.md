# Blogger Theme

A modern, responsive Blogger theme designed for mobile and Android apps, integrating:

- YouTube Channel & Video Feed
- Blog Posts with Dynamic Loading
- Social Media Links (Instagram, Facebook, etc.)
- Floating YouTube Button
- Clean UI with Logo Branding

![AppDateGH Logo](https://debeatzgh.wordpress.com/wp-content/uploads/2025/08/createamoderntech-inspiredlogoforadigitalcontenthubcalledappdategh4933013559151235986.jpg)

## 🔧 Features

- 📱 **Mobile & Android WebView Friendly**
- 🎥 **YouTube Integration** – Show latest 4 videos
- 📝 **Blogger Feed** – Dynamically pulls latest blog posts
- 📎 **Social Media** – Instagram, Facebook, and more
- 🎯 **Floating Action Button** – Quick access to YouTube or other CTAs
- 🌓 **Modern Look** – Sleek design with soft shadows and responsive layout
